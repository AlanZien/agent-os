# Global Standards

Standards universels applicables à tout le code AgentOS Tracker.

## Fichiers

| Standard | Usage |
|----------|-------|
| `global-standards.md` | Conventions de code, naming, structure |
| `security.md` | OWASP, auth, validation, secrets |
| `error-handling.md` | Gestion des erreurs, messages utilisateur |
| `api-contracts.md` | Contrats d'API, types partagés |
| `git-workflow.md` | Conventions Git, branches, commits |
| `ci-cd-devops.md` | CI/CD, déploiement |
| `environments.md` | Configuration des environnements |

## Chargement

Ces standards sont chargés automatiquement selon le contexte de travail.
Voir `orchestration.yml` dans chaque spec pour les standards spécifiques.
